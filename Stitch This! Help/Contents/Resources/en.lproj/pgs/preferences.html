Preferences [⌘,]

Allows you to specify the settings you'll use the most when using the app - which then means you need fewer steps to create a project.

The Restore Shipped Defaults button will, after confirmation, reset all Preferences back to the values they were set to when the app shipped.

Once you've entered them and clicked the Save button, they take immediate effect in the main app.

 The first pane, Symbols, enables you to define your own symbol set(s).
You may only wish to define one set, if there's only you stitching; but if your partner is also a keen stitcher, she or he may well prefer a different set of symbols to yours. It just allows you the flexibility.

To build a set, start by clicking the + button alongside the User dropdown. This enables you to enter the user's name, and click OK - which will cause that name to appear in the User dropdown, and present you with an empty symbol set (the grey region on the right of this pane). To build your set, double-click a symbol from the white pool of available symbols on the left, and that symbol will be copied across to the symbol set. To remove a symbol, double-click it in the symbol set, and it will be returned to the pool. Just keep repeating this for subsequent symbols to add them to the set. It won't allow you to enter past 50 symbols.

You may notice that as you double-click symbols, they disappear from the pool of available symbols - but so do others. These are the ones deemed visually similar to the one selected, and are therefore eliminated, to prevent you from building a symbol set which might have easily confused entries (e.g. '+' and '÷').

These symbols are auto-saved as you go. Clicking the Save button will also save the selected Set.

To remove a User's symbol set, simply click the '-' button alongside the User dropdown.
N.B. You can't delete the Default user - that is supplied by the app, and used if no user has been set up, or if you choose it when generating.

You can now select which user's symbol set should be used when generating output for an image. The one you leave selected in the User dropdown will become the Preference.
The second pane, Regional, enables you to set up regional preferences, such as whether you want to work with inches or centimetres, your choice of default stationery - i.e. A4 or Letter, and your choice of date format - i.e. European (DD/MM/YY) or U.S. (MM/DD/YY).

 The third pane, Materials, enables you to set up preferences for choice of materials, and is subdivided into two groups:
Fabric, which enables you to specify your favourite canvas and turnings allowance;
Strands, the number of strands you intend to work with, and
Allowance for turnings, or the amount of unused canvas around your finished work

Threads, where you can choose which manufacturer and thread range you prefer.
You can choose from Anchor, DMC, Madeira or Lecien Cosmo stranded cottons, or Anchor or DMC Tapestry wools.
The last pane, Output, enables you to choose preferences for what kind of output you would normally use:
Output charts in: which enables you to select B&W or Colour.
Thread card height: you can specify the height of your thread organiser cards, if you plan on using them.
'Date Started' prompt? you can choose whether a prompt to enter the date you started each page appears on the charts or not.
Output:Here you can choose from the range of print options for your project. Please see the Generate option for more detailed explanations of these selections.

Once specified, these Preferences will then be the default values for these fields whenever you use the app.
